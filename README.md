# 工作组智能体 (Workgroup Agent)

该仓库托管了**工作组智能体**，这是一个旨在支持团队运营、项目管理和日常工作流的智能助手。

## 核心目的

- **上下文感知**: 保持对公司和团队背景的深入理解。
- **技能执行**: 执行预定义的技能并随着时间的推移学习新技能。
- **自我进化**: 通过交互不断改进其能力。

## 快速开始

详见 [ARCHITECTURE.md](./ARCHITECTURE.md) 以获取系统结构的详细拆解。

## 目录概览

- `core/`: 系统逻辑、配置和可执行技能。
- `memory/`: 活跃上下文和过去交互的归档存储。
- `prompts/`: 系统提示词和特定任务模板。
- `outputs/`: 生成的产物和交付成果。
