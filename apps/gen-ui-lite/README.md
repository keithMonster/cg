# ⚡ GenUI Lite: 自编程 UI 编辑器

> **应用状态：活跃 | 版本：v1.2.0**

## 1. 应用愿景 (Vision)

GenUI Lite 是“数字超我”套装中的生产力工具，旨在直接在浏览器中通过对话生成并预览 Alpine.js 驱动的 UI 组件。

## 2. 技术规格 (Specifications)

- **架构**: Single File App (Zero-Build)
- **核心库**: Alpine.js v3
- **样式库**: Tailwind CSS v4 + Shoelace v2.12
- **设计系统**: ODS v1.0 (已对齐)

## 3. 关联 Prompt

- `frontend_application_architect`: 负责整体业务逻辑。
- `gen_ui_lite_architect`: 负责 UI 构建。
