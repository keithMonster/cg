# 系统与项目概览归档

## 已有系统

### 1. 帷幄 (Weiwo)

- **定位**：多页面合集平台，包含 CG Flow、QMS 等子业务。
- **技术栈**：
  - 前端：Vue 3 + Vite
  - 后端：Node.js
  - 基础设施：Docker + MySQL
- **核心功能**：行动点、画布等。

### 2. wflow

- **定位**：二次开发的低代码平台（公司购买）。
- **技术栈**：
  - 前端：Vue 2
  - 后端：Java
- **核心功能**：自定义表单与流程、AI 报告（自研二次开发）。

### 3. abi

- **定位**：低代码平台（公司购买）。
- **用途**：搭建图表、看板及自定义页面。

### 4. dataease

- **定位**：低代码平台（公司购买）。
- **用途**：搭建图表及可视化看板。

### 5. CG notes

- **定位**：笔记编写与发布系统。
- **技术栈**：
  - 前端：Vue 3
  - 后端：Node.js
- **核心功能**：语音转文字、AI 总结、周报生成。

### 6. CG Tender

- **定位**：面向投标团队的智能协作平台。
- **核心功能**：招标文件解读、风险识别、协同分析、知识沉淀。

## 部门基线项目分类

### OT 项目 (Operational Technology)

- 涉及领域：自动化、机器人、机械臂、自动焊等。

### IT / AI / 业务项目

- **信息化**：安灯系统、原材料管理、供应链。
- **流程与效率**：wflow、看板、QMS、CG Flow。
- **智能体与 AI**：CG notes、CG Brain（AI 应用、知识库、智能体）。
