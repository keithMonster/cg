# 智能化中心人员评价标准 (2024)

## 1. 红绿灯定位导向图 (Red-Green Light Matrix)

通过 **素质 (Competence)** 与 **绩效 (Performance)** 两个维度交叉判定红绿灯定位。

| 红绿灯定位 | 评价组合 (素质+绩效)              | 人才定位          | 人力资源关注度                | 介入/发展方式                        |
| :--------- | :-------------------------------- | :---------------- | :---------------------------- | :----------------------------------- |
| **1**      | 高素质 + 高绩效                   | **超级明星**      | 个体性高度关注 (继任者)       | 快速晋升、轮岗、授权、薪酬领先       |
| **2+**     | 中素质 + 高绩效 / 高素质 + 中绩效 | **核心骨干**      | 个体性高度关注 (高潜力)       | 谨慎规划晋升、轮岗、薪酬竞争性       |
| **2**      | 中素质 + 中绩效                   | **中坚力量**      | 整体性一般关注                | 重点开发培训，使其转化成明星         |
| **3**      | 高素质 + 低绩效 / 中素质 + 低绩效 | **潜力/业绩不佳** | 个体性重点关注 (判断是否转岗) | 必要时转岗，明确改进要求，不改则降级 |
| **4**      | 低素质 + 高绩效 / 低素质 + 中绩效 | **表现/素质不佳** | 一般关注 (判断培养潜力)       | 给予认可，减少管理职责，可考虑剥离   |
| **5**      | 低素质 + 低绩效                   | **失败者/淘汰**   | 重点关注 (判断是否淘汰)       | 尽快剥离出组织                       |

## 2. 素质评价标准 (Competence Rules)

| 素质项   | 定义                         | 合格 (2-3 分)                    | 优秀 (4-5 分)                    | 卓越 (6-7 分)                      |
| :------- | :--------------------------- | :------------------------------- | :------------------------------- | :--------------------------------- |
| **诚信** | 不怀恶意，与人为善，诚信正直 | 言行合一：遵守承诺               | 助人为乐：坚持原则，真心助人     | 止恶扬善：制止错误，恪守奉公       |
| **责任** | 面对困境主动承担，坚持目标   | 不言放弃：面对困难不轻言放弃     | 超越自我：主动承担失败，自我激励 | 树立标杆：强目标感，影响团队意志   |
| **协同** | 以他人需求为指南，服务意识   | 积极响应：及时响应他人需求       | 100%满意：专业处理，被他人信赖   | 超越期望：预测深层需求，主动出击   |
| **创新** | 保持开放心态，由内而外迭代   | 接受适应：理性对待变化，优化建议 | 推陈出新：主动更新知识/思维/方法 | 突破创新：前瞻性能量，推动组织进步 |

## 3. 业绩计分标准 (Performance Metrics)

| 定义           | 2-3 分 (合格)            | 4-5 分 (优秀)          | 6-7 分 (卓越)              |
| :------------- | :----------------------- | :--------------------- | :------------------------- |
| **工作贡献度** | 岗位工作贡献勉强达到预期 | 岗位工作贡献达到预期   | 岗位工作贡献超出预期       |
| **工作及时性** | 部分能按时完成，偶尔拖延 | 大部分情况下能按时完成 | 工作都能及时或提前完成     |
| **工作质量**   | 工作质量勉强符合要求     | 工作质量符合期望       | 工作质量超出预期，完全胜任 |

## 4. 潜力评价维度 (Potential Standards)

潜力分为三项标准，每项包含 7 个行为项。

- **学习敏捷度**: 自信但不自大、从错误中纠偏、敢于尝试变革、掌握新领域知识、困难环境镇定等。
- **职业抱负**: 设立高难度目标、接受挑战任务、寻求职业发展、影响激励他人、具备全局观等。
- **敬业度**: 期望长久服务、聚焦结果目标、奉献精神、主人翁意识、付出额外努力等。

### 评分规则：

- **低 (1 分)**: 满足 0-3 个行为要求
- **中 (2 分)**: 满足 4-5 个行为要求
- **高 (3 分)**: 满足 6-7 个行为要求
