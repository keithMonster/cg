# 公司与团队背景

## 公司信息

**名称**: 四川川锅锅炉有限责任公司 (Sichuan Chuanguo Boiler Co., Ltd.)
**背景**: 始建于 1968 年，由军工企业转产成立。北京建龙重工集团有限公司（世界 500 强）的全资子公司。
**定位**: 工业绿色能源清洁高效利用的引领者。专业从事高效节能锅炉及辅助设备的研发、设计、制造、安装、运行、服务于一体的高新技术企业。
**核心能力**: A 级锅炉制造、压力容器设计制造资质，拥有国家企业技术中心。具备 350MW 常规电站锅炉、核级容器及大型化工容器制造能力。
