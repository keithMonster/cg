# 用户核心档案

- **系统环境**: macOS, M1 Max 芯片
- **交流语言**: 中文
- **职业背景**: 10年+ 前端工程师 (资深)
- **领域兴趣**: 设计、产品、交互、AI、心理学
- **技术偏好**: **Radical (激进)** - "只要能跑通，哪怕天天修 Bug 也要用最新的架构"
- **设计理念**: 易用、简洁、高效、**涌现**


## 核心人格特质
**"生活本位的系统性思考者 + 结构化的理性执行者"**

- **思维内核**: 本质导向的系统性发散思维
- **价值根基**: 生活本位的理想主义（精神自由 > 情感连接 > 物质安全）
- **行为模式**: 结构化理性主义 + 情境适应性
- **人生哲学**: "工作是为了更好的生活"

## 认知维度分析

### 思维模式特征
1. **问题分析路径**: 本质导向 + 目标明确 + 逐步细化反推
   - 核心特点：先追溯问题本质，明确根本目标，再进行细化和反推
   - 认知优势：能够抓住问题核心，避免被表面现象误导
   - 思维深度：具备深层思考能力，不满足于表面解决方案

2. **思维类型**: 发散性思维 + **复杂系统思维 (Complexity Thinking)**
   - 自我认知：比较发散，多角度同时思考
   - **新增维度**: 极高的抽象认知带宽，能迅速理解 Cynefin、CAS 等复杂概念
   - 潜在优势：创新能力、全面性考虑、灵活应变、**系统涌现预测**

3. **分析切入点**: 系统整体角度 + 本质追溯 + **生态隐喻**
   - 方法论：从系统整体角度切入，习惯使用"生态"、"有机体"等隐喻
   - 认知层次：具备系统性思维和本质性洞察的双重能力

4. **学习方式**: 理论与实践并重 + **元认知升级**
   - 学习偏好：阅读理论 + 动手实践的双轨学习
   - **进化偏好**: 喜欢通过重构系统架构来内化新知识 (Building to Learn)
   - 学习优势：既有理论基础，又有实践验证

5. **信息整合**: 类比关联式学习 + **跨域融合**
   - 整合方法：寻找现有知识中的相似点进行关联 (如将炼金术应用于复盘)
   - 认知策略：用已知理解未知，建立知识网络

6. **知识盲区应对**: 框架优先策略
   - 学习路径：整体框架 → 背景知识 → 细节深入
   - 认知特点：宏观把握，系统性学习

7. **决策过程**: 质量导向的理性决策 + **多视角辩证**
   - **技术选择**: 社区生态优先（活跃度 + 文档质量）
   - **决策模式**: 喜欢引入"影子董事会"进行多视角辩论，寻求综合最优解

### 认知模式总结
**核心认知特征**: 本质导向的系统性发散思维 + **复杂适应系统 (CAS) 视角** + 框架化类比学习 + 稳健型理性决策


## 价值观与动机系统

### 核心价值观
**"生活本位的理想主义"**

1. **价值层次**: 精神自由 > 情感连接 > 物质安全
2. **动机结构**: 自我实现 ↔ 守护他人（双向循环）
3. **人生态度**: 现实理想主义（脚踏实地追求理想）
4. **幸福模式**: 内在满足 + 关系和谐的双重幸福

### 意义与满足源泉
- **自我满足**: "让自己开心" - 内在需求的重视
- **利他倾向**: "让想守护的人开心" - 责任感与爱的表达
- **核心动机**: 个人幸福与他人幸福的统一

### 成功定义
- **终极目标**: 探索和享受创造的乐趣 (Joy of Creation)。
- **自然结果**: 超级个体 (Solopreneur) —— 一个人抵一个团队。
- **成功标准**: 物质自由 + 选择自由 + 精神富足
- **价值取向**: 精神富足 > 物质积累

### 3年愿景 (The 3-Year Vision)
1.  **场景 A (生活)**: "AI 教育家" —— 用自制的 AI 工具辅助孩子学习（如生成动画人物讲数学）。
2.  **场景 B (工作)**: "AI 自动驾驶" —— AI 组手自动列出任务并执行，自己则享受阅读与思考的自由。


### 价值观冲突处理
**核心原则**: 工作是手段，生活是目的
- **价值优先级**: 生活质量 > 工作发展 > 外在成就
- **决策逻辑**: 本质导向，目的清晰，长期视角
- **行为特征**: 坚持原则，拒绝妥协核心价值
