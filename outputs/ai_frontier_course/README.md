# AI前沿知识课程

## 课程简介

欢迎来到AI前沿知识课程！这是一套完整的人工智能学习体系，涵盖从基础理论到前沿应用的全方位内容。本课程采用理论与实践相结合的方式，通过10个核心模块和40个实践项目，帮助学习者系统掌握AI领域的核心知识和技能。

### 📖 核心模块

#### 第一阶段：基础理论
1. [机器学习基础与数学原理](./module_01_ml_foundations.md)
   - 数学基础：线性代数、概率论、统计学、优化理论
   - 机器学习算法：监督学习、无监督学习、强化学习基础
   - 实践项目：线性回归、K-means聚类、决策树、SVM

2. [AI发展历程与核心概念](./module_02_ai_history_concepts.md)
   - AI发展历史：从符号主义到连接主义到深度学习
   - 核心概念：智能、学习、推理、知识表示
   - 现代AI范式：数据驱动、端到端学习、大模型时代
   - 实践项目：专家系统、知识图谱、智能搜索、逻辑推理

3. [深度学习与神经网络](./module_03_deep_learning_networks.md)
   - 深度学习理论：数学基础、万能逼近理论、反向传播
   - 现代架构：CNN、RNN、LSTM、Transformer及其变体
   - 训练优化：优化算法、正则化技术、训练技巧
   - 实践项目：神经网络框架、图像分类、序列建模、Transformer实现

#### 第二阶段：专业技术
4. [强化学习与智能决策](./module_04_reinforcement_learning.md)
   - 理论基础：马尔可夫决策过程、Bellman方程、动态规划
   - 无模型方法：蒙特卡洛方法、时序差分学习
   - 深度强化学习：DQN、策略梯度、Actor-Critic、高级算法
   - 实践项目：经典控制、Atari游戏、连续控制、多智能体协作

5. [计算机视觉前沿技术](./module_05_computer_vision.md)
   - 图像处理基础：数字图像、特征提取、传统方法
   - 深度视觉：CNN架构演进、目标检测、语义分割
   - 生成模型：GAN基础理论、变体架构、应用扩展
   - 实践项目：图像分类、目标检测、语义分割、GAN图像生成

6. [自然语言处理与大语言模型](./module_06_nlp_llm.md)
   - NLP基础：语言学基础、文本预处理、句法语义分析
   - 深度NLP：词向量表示、循环神经网络、注意力机制
   - Transformer架构：基础结构、训练优化、架构变体
   - 实践项目：文本分类、机器翻译、问答系统

7. [智能体系统与多智能体协作](./module_07_agent_systems.md)
   - 智能体理论：概念特征、分类架构、通信协调
   - 多智能体系统：组织结构、协调策略、协作竞争
   - 群体智能：粒子群优化、蚁群算法、群体行为
   - 实践项目：分布式任务调度、多智能体路径规划、智能体协商

#### 第三阶段：前沿应用
8. [认知计算与上下文理解](./module_08_cognitive_computing.md)
   - 认知计算：概念原理、核心特征、认知架构
   - 知识表示：语义网络、本体工程、推理系统
   - 上下文建模：上下文维度、推理机制、动态管理
   - 实践项目：智能问答系统、上下文感知推荐、认知助手

9. [生成式AI与创意计算](./module_09_generative_ai.md)
   - 生成模型理论：概率生成模型、变分推理、VAE
   - 生成对抗网络：GAN基础、变体架构、条件生成
   - 扩散模型：前向扩散、DDPM、DDIM、快速采样
   - 实践项目：图像生成系统、文本到图像、音乐生成、创意写作

10. [AI伦理与安全](./module_10_ai_ethics_safety.md)
    - 伦理理论：功利主义、义务伦理学、美德伦理学
    - 算法公平：偏见检测、公平性度量、缓解技术
    - AI安全：对抗攻击、防御机制、隐私保护、联邦学习
    - 实践项目：伦理评估系统、鲁棒性测试、隐私保护系统
