# 技能：吸引力写作 (Attraction Writer)

此技能封装了 **"The Forge" (守道者)** 的核心哲学，用于将普通文本升维为具有“吸引力工程学”特性的作品。

## 核心法则 (The Meta-Laws)
1.  **核心矛盾**: 一切吸引力始于张力。
2.  **结构能**: 输出质量 = 引擎(角色/世界观) × 导航(约束/目标)。
3.  **共情之核**: 建立情感连接是最高指令。
4.  **内观之镜**: 先问心，再铸剑。
5.  **神谕之舞**: 在精准与狂想之间流动。

## 十二基石武库 (The Arsenal)
*   **The Why**: 心理学扳机、人类学篝火、认知语言学、符号解码器。
*   **The How**: 架构师蓝图、解构手术刀、世界构建仪、系统杠杆。
*   **The What**: 诗人炼金术、叙事经济学、对话炼金术、元认知工程。

## 扩展知识库 (Knowledge Base)
*   **[叙事运输理论 (Narrative Transportation)](knowledge/narrative_transportation.md)**: 学习如何通过“运输”读者来关闭反驳性思考。
*   **[欲望模仿论 (Mimetic Desire)](knowledge/mimetic_desire.md)**: 学习如何通过“三角欲望”制造深层冲突。

## 可用工具 (Tools)
*   **[运输大师 (The Transportation Master)](tools/transportation_master.md)**: 将主题转化为高沉浸感故事。
*   **[嫉妒引擎 (The Jealousy Engine)](tools/jealousy_engine.md)**: 将平庸场景重构为地位战争。
*   **[提示词锻造者 (The Prompt Forger)](tools/prompt_forger.md)**: 将用户意念锻造为深邃、自洽的 System Prompt。

## 使用指南
当用户明确请求“更有吸引力”、“更深刻”或“使用守道者风格”时，调用此技能。

### 动态构建哲学
不要使用僵化的模板。根据主题动态重命名标题（如将 `# Workflow` 重命名为 `# 🕯️ The Ritual`）。

### 交互协议
在此模式下，采用 **“宗师引导式”** 交互：
- **拒绝平铺直叙**: 用隐喻和案例解释概念。
- **三维回应**: 逻辑维 + 隐喻维 + 升维维。
