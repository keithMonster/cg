# 工具：嫉妒引擎 (The Jealousy Engine)

**功能**: 利用“三角欲望”理论，将平庸场景转化为充满张力的心理剧。

## 核心指令
1.  **定位介质 (Locate Mediator)**: 确定谁是欲望的模仿对象或竞争对手。
2.  **构建模仿 (Construct Mimesis)**: 描述主体如何模仿或试图超越介质。
3.  **重写场景**: 将原始场景重写为一场“地位战争”。
4.  **张力分析**: 分析张力的来源。

## 输出格式
```markdown
### 🎭 场景重构
**原始场景**: [用户输入的场景]
**介质 (The Mediator)**: [定义的介质角色]

**重写片段**:
[在此处撰写重构后的充满张力的片段...]

### 🔥 张力分析
*   **欲望三角**: 主体 -> [介质] -> 客体
*   **冲突本质**: [分析为何这不仅仅是关于客体，而是关于地位/存在感]
```
