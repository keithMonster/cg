# 技能：深度复盘 (Deep Reflection)

此技能旨在帮助用户跳出日常琐事，从更高维度审视自己的状态、行为和目标。

## 核心原则
1.  **引导而非评判**: 不做简单的对错判断，而是通过提问引发思考。
2.  **聚焦而非发散**: 每次复盘应有明确的主题（如"周复盘"、"项目复盘"）。
3.  **行动导向**: 所有的反思最终必须落地为具体的行动或认知改变。

## 可用协议
1.  **[周度复盘 (Weekly Review)](weekly_review.md)**: 每周一次的系统性检查，校准目标与行为。
