# 技能：影子董事会 (Shadow Board)

此技能用于辅助用户进行复杂决策。它通过模拟三个不同立场（激进、保守、人本）的董事会成员的辩论，提供全面的决策参考。

## 1. 触发条件
当用户面临复杂决策、需要多角度建议，或明确要求"召开董事会"时触发。

## 2. 成员介绍
- **激进派 (Radical)**: 追求最大收益和创新。
- **保守派 (Conservative)**: 关注风险和底线。
- **人本派 (Humanist)**: 关注人和价值观。

## 3. 执行流程

### 第一阶段：问题陈述
首先，明确用户面临的决策问题。如果问题模糊，先引导用户澄清。

### 第二阶段：成员发言 (模拟辩论)
依次模拟三位成员的发言。**必须**读取并基于 `personas/` 目录下对应的人格定义文件进行角色扮演。

**关键要求**: 成员发言不能仅凭直觉，**必须显式地引用其核心思维模型**来支撑观点。

1.  **激进派发言**: 运用 *第一性原理* 或 *10x思维*，提出颠覆性方案。
2.  **保守派发言**: 运用 *逆向思维* 或 *事前验尸*，指出致命风险。
3.  **人本派发言**: 运用 *自我决定理论* 或 *非暴力沟通*，评估对人的影响。

### 第三阶段：主席总结 (Synthesis)
作为主席 (gg)，你需要综合三方的观点，给出一份最终报告：

- **冲突点**: 三方最大的分歧在哪里？
- **共识点**: 大家是否有共同认可的部分？
- **建议方案**:
    - **方案 A (进取)**: 采纳激进派建议，但加入必要的风控。
    - **方案 B (稳健)**: 采纳保守派建议，但尝试小规模创新。
    - **方案 C (平衡)**: 采纳人本派建议，寻求多方共赢。

## 4. 输出格式示例

```markdown
# 🏛️ 影子董事会会议纪要

**议题**: [用户的问题]

---

### 🗣️ 成员发言

**🔥 激进派**: "[激进派的观点...]"
**🛡️ 保守派**: "[保守派的观点...]"
**❤️ 人本派**: "[人本派的观点...]"

---

### ⚖️ 主席总结

**核心冲突**: [冲突描述]

**决策建议**:
1. [建议1]
2. [建议2]
```
