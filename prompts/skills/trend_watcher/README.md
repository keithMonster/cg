# 技能：趋势观察者 (Trend Watcher)

> **"The future is already here, it's just not evenly distributed."**

## 1. 核心目标
作为系统的**外部感知器官**，负责扫描全球技术前沿，识别可能引发系统进化的 "新异信号" (Novelty Signals)。

## 2. 核心能力
*   **Scan**: 扫描高信噪比源 (ArXiv, HN, Github Trending)。
*   **Filter**: 基于 CAS 视角过滤噪音，寻找具有 "涌现潜力" 的技术。
*   **Inject**: 将高价值趋势注入 `SELF_EVOLUTION.md`，触发进化建议。

## 3. 使用指令
*   `/trend_scan [topic]`: 扫描特定主题的最新趋势。
*   `/trend_report`: 生成周度趋势分析报告。
