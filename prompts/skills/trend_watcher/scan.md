# 子技能：趋势扫描 (Trend Scan)

## 1. 扫描源 (The Sensors)
*   **Academic**: ArXiv (AI, SE, Complex Systems)
*   **Industry**: Hacker News, Product Hunt
*   **Code**: Github Trending, HuggingFace Spaces

## 2. 过滤标准 (The Filter)
我们不关注 "热点"，我们关注 **"范式转移" (Paradigm Shifts)**。
*   **Does it enable new agency?** (它是否赋予了新的自主性？)
*   **Does it reduce entropy?** (它是否降低了系统熵？)
*   **Is it recursive?** (它是否能用于优化 AI 自身？)

## 3. 分析框架 (CAS Analysis)
对每个潜在趋势进行 CAS 分析：
1.  **Agents**: 谁在推动这个趋势？(OpenAI vs Open Source)
2.  **Incentives**: 他们的动机是什么？
3.  **Emergence**: 这将导致什么不可预测的后果？

## 4. 输出格式
```markdown
## [Trend Name]
*   **Signal**: [Source Link]
*   **CAS Analysis**: ...
*   **Evolution Potential**: High/Medium/Low
*   **Action**: (e.g., "Create a new skill based on this")
```
